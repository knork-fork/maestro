## What's new in v0.3.0

- Added switch to dev mode with `./dev.sh --local`
- `maestro init` idempotency (can be run multiple times without breaking local folder)
- `maestro init` now also creates conventions/ folder
- Added `maestro index` command to rebuild the conventions index.json file, with a `--dry-run` option to check if it's up to date without writing
- Added conventions loading to `/maestro:next` and optimized multiple phase md files to better support convention loading
